## Accessing settings

Click your account name or company logo in the top-left of the web app and select **Settings**.

## Company settings

Under **Settings → General**, you can update:

- **Company name** — Displayed in the app and in reports
- **Time zone** — Used for scheduling and call timestamps
- **Business hours** — Used by the AI Voice Agent to determine when to route calls to live staff

## Notifications

Under **Settings → Notifications**, configure when CraftFlow sends you alerts:

| Notification | Description |
|-------------|-------------|
| Call scored below threshold | Get notified when a call fails your scorecard |
| New coaching assignment | Notify the assignee when a coaching task is created for them |
| Recording processed | Notify when a field recording finishes processing |
| New voice agent booking | Notify when the AI books an appointment |

Notifications can be sent via **email** or **in-app**.

## API keys

If you're building a custom integration, go to **Settings → Integrations → API Keys** to generate an API key. See the [API Reference](/api/introduction) for documentation on what you can build.

## Billing

Billing is managed under **Settings → Billing**. You can view your current plan, usage, and invoices. To change your plan or add seats, contact [support@craftflow.com](mailto:support@craftflow.com).
