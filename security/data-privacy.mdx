## What data CraftFlow stores

CraftFlow stores the following categories of data on behalf of your business:

| Category | Examples | Purpose |
|----------|----------|---------|
| **Customer data** | Names, addresses, phone numbers | Synced from your CRM/FSM for context in CraftFlow |
| **Call recordings** | Audio files from CSR and field calls | Transcription, scoring, and coaching |
| **Transcripts** | Text of recorded calls | AI analysis and search |
| **User data** | Your team's names, emails, roles | Authentication and access control |
| **Usage data** | Login times, feature usage | Product improvement and support |

## Data encryption

- **In transit:** All data is encrypted using TLS 1.2 or higher
- **At rest:** Call recordings stored in S3 are encrypted using AES-256. Database data is encrypted at rest using AWS RDS encryption.

## Data retention

| Data type | Default retention |
|-----------|------------------|
| Call recordings (audio) | 12 months |
| Transcripts | 12 months |
| Scorecard results | 24 months |
| User account data | Until account deletion |

You can request early deletion of specific recordings or all data by contacting [support@craftflow.com](mailto:support@craftflow.com).

## Subprocessors

CraftFlow uses the following third-party services to process data:

| Subprocessor | Purpose |
|-------------|---------|
| Amazon Web Services | Hosting, storage |
| AssemblyAI | Call transcription |
| OpenAI | AI analysis and coaching insights |
| Pinecone | Vector search for knowledge base |
| Sentry | Error monitoring (no PII in error reports) |
| PostHog | Product analytics (anonymized usage data) |

## Your rights

As a CraftFlow customer, you can:

- **Access** your data by contacting us or via the API
- **Delete** your data by submitting a deletion request to [support@craftflow.com](mailto:support@craftflow.com)
- **Export** call recordings and transcripts from within the app

For your end customers (homeowners), requests to access or delete their data should be submitted through your own privacy process. Contact us if you need help fulfilling those requests.

## Data residency

All CraftFlow data is stored and processed in the **United States**. We do not currently offer EU or other regional data residency options.
